# Renegade Drums and Percussions

Simple SEO-friendly React site for Renegade Drums and Percussions — a local Philippine marching band in Cebu City (double drums, bass & lyre, bugle).

## Run locally

```bash
cd renegade-drums
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
